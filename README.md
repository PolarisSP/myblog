# simple-blog

#### 介绍
一个极简主义Blog网站

#### 运行
如果你想运行该项目，你首先应该 Clone 项目，
然后进入项目目录 输入 npm install,安装项目依赖
之后再输入 npm run docs:dev 该项目将会在你本地开启一个服务器，并运行在 8080 端口
